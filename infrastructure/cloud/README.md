# Cloud Planning Boundary

Phase 01 chỉ bảo đảm application Cloud-ready thông qua stateless API container, static Web artifact, environment validation và health/version endpoints.

Các quyết định Cloud provider, region, managed MongoDB, object storage, registry, domain/TLS, secret manager, monitoring và budget phải được chốt trước Phase 07. Không lưu credential Cloud trong thư mục này.
