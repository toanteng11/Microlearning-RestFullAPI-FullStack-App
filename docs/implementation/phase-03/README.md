# Phase 03 - Classroom Management

## Trạng thái

`Planned`. Kế hoạch chi tiết được baseline sau Phase 02.

## Mục tiêu sơ bộ

Cho phép Teacher tạo/quản lý Classroom, công khai Class Code hoặc Invite Link và xem roster. Student đã đăng nhập có thể join bằng Code hoặc Link; QR không thuộc phạm vi.

## Capability chính

- Classroom lifecycle và ownership.
- Enrollment policy, duplicate prevention và capacity/status checks.
- Join by Class Code và Join by Invite Link.
- Teacher roster và Student classroom list.
- Audit/authorization cho hành động enrollment nhạy cảm.

## Dependency

Authentication, active account policy và RBAC từ Phase 02.
