# Phase 04 - Learning Content

## Trạng thái

`Planned`. Kế hoạch chi tiết được baseline sau Phase 03.

## Mục tiêu sơ bộ

Cho phép Teacher xây dựng Course, Module/Topic, Micro Lesson, resource và announcement; quản lý draft/published/archived lifecycle và deadline riêng cho từng learning activity.

## Capability chính

- Course/module/lesson CRUD và ordering.
- Content publish/unpublish/archive rules.
- Lesson player và navigation Previous/Next/Back.
- Activity deadline và deadline history foundation.
- Student chỉ thấy content đã publish trong Classroom đã tham gia.

## Dependency

Classroom ownership/enrollment từ Phase 03; quyết định object storage nếu triển khai upload media.
