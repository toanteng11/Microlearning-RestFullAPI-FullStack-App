# Phase 06 - Reporting and Analytics

## Trạng thái

`Planned`. Kế hoạch chi tiết được baseline sau Phase 05.

## Mục tiêu sơ bộ

Tổng hợp dữ liệu học tập thành Student To-do, own progress và Teacher course dashboard với roster, completion, process score và ranking từ cao xuống thấp; cung cấp Admin report/audit cơ bản.

## Capability chính

- Student Dashboard To-do cho công việc chưa hoàn thành.
- Learning progress và course completion summary.
- Teacher roster/progress/process score/ranking.
- Recalculation khi submission, grade hoặc deadline thay đổi.
- Reporting access control và privacy tests.

## Dependency

Enrollment, content, assessment, grading và deadline data từ Phase 03-05.
