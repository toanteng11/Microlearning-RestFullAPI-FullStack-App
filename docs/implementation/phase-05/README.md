# Phase 05 - Assessments and Grading

## Trạng thái

`Planned`. Kế hoạch chi tiết được baseline sau Phase 04.

## Mục tiêu sơ bộ

Triển khai Quiz, Assignment, Submission, Grade và Feedback, bao gồm media tùy chọn trong câu hỏi khi storage policy sẵn sàng và chức năng Teacher reset deadline có lý do/lịch sử.

## Capability chính

- Quiz/question/answer option và attempt lifecycle.
- Optional image/video question media theo policy.
- Assignment, submission và late status.
- Auto/manual grading và private feedback.
- Deadline reset/recalculation, reason, history và authorization.

## Dependency

Course/content lifecycle từ Phase 04; media/storage policy; grading rules và acceptance criteria.
